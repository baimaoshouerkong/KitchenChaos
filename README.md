## 创新
如果就是跟着教程完成的游戏，那就没有了游戏的灵魂，毕竟如果做出来的游戏自己都玩不下去，那么可真的就是失败了

### 目标
为了让游戏好玩一些，就要给玩家一点目标，比如每天或者每周扣钱，或者一些buff上的奖励

### 事件
一个游戏必须要有特殊的事件，而这些事件必须伴随玩家的游玩
